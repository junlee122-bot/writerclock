# 변경 기록

## 미출시

- 활판 인쇄물 콘셉트로 UI 전면 개편: 글래스모피즘 카드·장식 제거, 시계 숫자와
  인용문·판권을 하나의 명조 체계로 통일
- Noto Serif KR 가변 폰트 서브셋(약 430KB)을 번들해 모든 플랫폼에서 동일한
  명조 조판 보장(OFL 1.1 고지 포함, 오프라인 캐시 등재)
- 두 자리 시(10–12시) 시계 숫자가 잘리던 버그를 컨테이너 기준 크기 계산으로 수정
- 라이트 테마 대비를 WCAG AA로 상향(본문·보조·강조 토큰 재조정), 다크 테마를
  따뜻한 잉크 톤으로 재구성, 브랜드 마크를 테마 무관 낙관 색으로 고정
- 현재/탐색 모드를 일급 상태로 표시: 탐색 중 배너와 ‘지금으로 돌아가기’,
  탐색 중 가짜 초·진행 표시 제거, 거치 모드 진입 시 현재 시각 복귀와
  유휴 3분 후 자동 복귀
- 검수용 메타데이터 칩을 본문에서 정보 대화상자로 이동(민감한 내용·오전오후
  미확정 경고만 각주로 유지), 인용 출처 옆 ‘문장 정보’ 바로가기 추가
- 분 전환 안무(문장 교차 전환), 저장 애니메이션, 토스트 전환과 ‘되돌리기’,
  전체 삭제 2단계 확인 추가
- −1분/+1분 길게 눌러 연속 이동, Shift+화살표 10분 이동, Esc 거치 종료,
  한국어 IME에서도 동작하는 단축키(N/S/D)와 끄기 설정 추가
- 시각 선택기를 오전/오후 한국어 표기로 통일, 설정 스위치·세그먼트 컨트롤
  커스텀 스타일링, 화면 어둡게 값 표시 추가
- 접근성 보강: 시계의 오전·오후 낭독, 자동 분 갱신의 스크린리더 소음 제거,
  영어 원문 lang 표기, 44px 터치 타깃, 강제 고대비 모드의 선택 상태 표시,
  동작 줄이기 시 번인 방지 이동 중단
- 짧은 가로 화면(폰 거치)에서 열 최소 너비 초과로 내용이 잘리던 문제 수정
- 별도 항목 8개의 1차 출전을 확인하고, 잘못된 오전 구절이던 18:44를 공식 저자 샘플의 명시적 오후 구절로 교체
- 출전 검토와 오전·오후 확실성을 분리하고 모호한 시간대 204개를 앱에 표시
- 번역 1,440개 전부의 원문 발췌와 재현 가능한 출전 감사·회귀 검증 추가

## 1.0.0 — 2026-07-13

- 상류 Author Clock 전체 자산을 WriterClock 저장소로 이식
- 1,440분 정밀 데이터의 정본과 생성 산출물을 분리하고 재현 검사 추가
- 부분 일치, 복합어, 여러 시각, 야간 오전/오후, 근사 시각 오탐 제거
- 오전/오후 단서 없는 12시간제 원문을 24시간 정밀 코퍼스에서 제외
- 근사 번역 17개를 문학 원문에 정확한 시각이 있는 항목으로 교체
- 시간 탐색, 고정 링크, 즐겨찾기 검색·입출력, 공유, 설치·업데이트 UX 추가
- 즐겨찾기 가져오기를 정본 데이터에 재검증하고 단일 문장 셔플을 비활성화
- 접근 가능한 대화상자, 키보드 이동, 반응형 도구막대와 동작 줄이기 지원
- 서비스 워커 갱신, GitHub Pages 배포와 PR 검증 워크플로 추가
- Tauri 데스크톱 앱을 원격 웹 래퍼에서 오프라인 자산 번들로 전환
- 영문 편집물 CC BY-NC-SA 2.5와 Pretendard OFL 1.1 고지를 저장소에 보존
- ESP32-S3 재현 빌드, 폰트 체크섬 검사와 다운로드 가능한 펌웨어 산출물 추가
