# 변경 기록

## 미출시

- 출전 후보 338개와 미매핑 281개를 전수 검토해 canonical 원문 행에 고정
- 별도 항목 8개의 1차 출전을 확인하고, 잘못된 오전 구절이던 18:44를 공식 저자 샘플의 명시적 오후 구절로 교체
- 출전 검토와 오전·오후 확실성을 분리하고 모호한 시간대 204개를 앱에 표시
- 번역 1,440개 전부의 원문 발췌와 재현 가능한 출전 감사·회귀 검증 추가

## 1.0.0 — 2026-07-13

- 상류 Author Clock 전체 자산을 WriterClock 저장소로 이식
- 1,440분 정밀 데이터의 정본과 생성 산출물을 분리하고 재현 검사 추가
- 부분 일치, 복합어, 여러 시각, 야간 오전/오후, 근사 시각 오탐 제거
- 오전/오후 단서 없는 12시간제 원문을 24시간 정밀 코퍼스에서 제외
- 근사 번역 17개를 문학 원문에 정확한 시각이 있는 항목으로 교체
- 시간 탐색, 고정 링크, 즐겨찾기 검색·입출력, 공유, 설치·업데이트 UX 추가
- 즐겨찾기 가져오기를 정본 데이터에 재검증하고 단일 문장 셔플을 비활성화
- 접근 가능한 대화상자, 키보드 이동, 반응형 도구막대와 동작 줄이기 지원
- 서비스 워커 갱신, GitHub Pages 배포와 PR 검증 워크플로 추가
- Tauri 데스크톱 앱을 원격 웹 래퍼에서 오프라인 자산 번들로 전환
- 영문 편집물 CC BY-NC-SA 2.5와 Pretendard OFL 1.1 고지를 저장소에 보존
- ESP32-S3 재현 빌드, 폰트 체크섬 검사와 다운로드 가능한 펌웨어 산출물 추가
