# 기여 안내

1. 이 저장소를 포크하고 작은 목적의 브랜치를 만듭니다.
2. `npm run check`와 `python scripts/build_ko_quotes.py --check`를 실행합니다.
3. UI 변경은 320px 모바일, 짧은 가로 화면, 키보드, 다크 모드와 오프라인을 확인합니다.
4. 데이터 변경은 정본 입력을 수정하고 생성기를 실행한 뒤 생성 파일도 함께 커밋합니다.

문학 인용을 추가할 때는 `DATA_LICENSE.md`의 필드를 모두 제공해야 합니다.
정확한 분을 말하지 않는 근사 표현이나 출전이 확인되지 않은 문장은 받지
않습니다. 한국어 번역은 원문 의미와 시각을 별도로 검토해 주세요.

PR 설명에는 변경 이유, 검증 명령, 눈에 보이는 UI 변경의 캡처, 데이터라면
출처 URL과 권리 근거를 포함합니다. 자동 생성 파일만 단독으로 수정하지 마세요.
