# Security Policy

## 지원 범위

`main`의 최신 버전과 최신 GitHub Pages 배포를 지원합니다. 오래된 포크,
수정된 펌웨어 이미지, 제3자 미러는 지원 범위 밖입니다.

## 비공개 신고

취약점에는 공개 이슈에 재현 정보를 올리지 말고 GitHub 저장소의
**Report a vulnerability** 기능이 보이면 그 채널로 영향, 브라우저·운영체제와
가능한 완화책을 보내 주세요. 메뉴가 아직 보이지 않으면 민감한 세부정보 없이
`[security contact]` 제목의 이슈만 열어 비공개 연락 채널을 요청해 주세요.
어느 경우에도 개인정보나 자격 증명을 첨부하지 마세요.

문학 콘텐츠의 권리·출전 오류는 보안 취약점은 아니지만, 긴급 비공개 처리가
필요한 경우 같은 채널을 사용할 수 있습니다.
